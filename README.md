## 📋 Overview

This repository contains all exercises, assignments, and the final project from the **MLOps Bootcamp** conducted by **Hamrah Aval (MCI)**. The bootcamp focuses on production-ready Machine Learning practices and tools.

### 🎯 Learning Objectives
- Master version control with Git & GitHub
- Containerize ML applications using Docker
- Track experiments and models with MLflow
- Build CI/CD pipelines for ML projects
- Implement model monitoring and logging
