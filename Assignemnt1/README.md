## 📋 Exercise Description

**In the name of God**

### Objective

The goal of this exercise is to **deploy a Machine Learning model**. More specifically, we had to:

1. Train a **Deep Learning model** for **binary image classification** (Dogs vs. Cats)
2. **Deploy** the trained model using **FastAPI** framework
3. Create a RESTful API that accepts an image and returns whether it's a dog or a cat
